Première Connexion & Interface
📚 Théorie Docker

🛠 Pourquoi Portainer ?

Portainer transforme les commandes Docker complexes en interface graphique intuitive, essentielle pour administrer un laboratoire multi-services.

🛠 Connexion Guidée
  • Maîtriser l'interface Portainer
  • Comprendre l'architecture CyberLab
  • Naviguer dans les sections principales
  • Observer les métriques système
🎯 Exercise 1.1: Connexion
Test de connectivité Portainer

Ouvrir Portainer : http://127.0.0.1:9000
